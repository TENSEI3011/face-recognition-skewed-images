🛡️ Anti-Spoofing / Liveness Detection
What this does: Before matching any face, the system checks whether it is a real living face or a presentation attack (printed photo, phone screen, video replay, or mask). Faces that fail the liveness check are immediately rejected as SPOOF DETECTED without running identity matching.   —   Uses passive texture + frequency analysis. No extra model download needed. Runs in ~3ms.
